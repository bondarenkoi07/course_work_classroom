Описание проекта
Данное веб-приложение является реализацией поставленной задачи в курсовой работе. Для решения поставленной задачи применялся стек технологий:фреймворк goji и веб-инструментарий gorilla для языка программирования GO, JQuery и SVG.js для javascript. Статический файлы обслуживается http-клиентом Nginx. Исходный код приложения выложен на гитхаб.

Goji framework
Goji - это мультиплексор HTTP-запросов, аналогичный net/http.ServeMux. Он сравнивает входящие запросы со списком зарегистрированных шаблонов и отправляет их в http.Handler, который соответствует первому совпадающему шаблону. Подробнее о фреймворке

SVG.js

Легкая библиотека для управления и анимации SVG. SVG.js не имеет зависимостей и стремится быть как можно более компактным, обеспечивая при этом почти полное покрытие спецификации SVG. Если вы еще не уверены, то вот несколько основных моментов. SVG.js работает быстро. Очевидно, не так быстро, как vanilla js, но во много раз быстрее, чем Snap.js Создание и управление SVG с помощью одного только JavaScript довольно многословно. В проекте использован для отрисовки схемы учебной аудитории.

Подробнее о фреймворке

Gorilla web-toolkit

Gorilla - это веб-инструментарий для языка программирования Go. В даном проекте использован пакет gorilla/websocket, имплементирующий протокол WebSocket, определенный в RFC 6455. В данном проекте использовался для непрерывного обмена данных между клиентом и бэкендом.
